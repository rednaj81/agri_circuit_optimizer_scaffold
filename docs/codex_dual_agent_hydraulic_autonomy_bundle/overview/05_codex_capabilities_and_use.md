# Como este bundle usa recursos do Codex

A automação proposta assume:

- subagentes / agentes especializados
- AGENTS.md
- skills
- orquestração via Codex MCP server
- possibilidade de trabalho em background/no cloud quando aplicável

A implementação concreta pode ser feita:
- via Codex CLI + MCP server + Agents SDK
- ou via workflow manual assistido no próprio Codex, seguindo as mesmas regras
