# Objetivos do sistema

Desenvolver um solucionador que sintetize automaticamente um circuito hidráulico agrícola,
incluindo topologia, componentes, medição e viabilidade hidráulica simplificada.

## O que o sistema deve otimizar

- custo estrutural
- número total de componentes
- cobertura de rotas obrigatórias
- adequação de medição
- atendimento de vazão mínima
- atendimento de dosagem mínima com erro máximo
- robustez hidráulica simplificada

## O que o sistema deve produzir

- arquitetura ótima
- lista de materiais (BOM)
- mapeamento de rotas atendidas
- atribuição de bombas e fluxômetros
- relatório de perdas e folgas
- esquema lógico de montagem
