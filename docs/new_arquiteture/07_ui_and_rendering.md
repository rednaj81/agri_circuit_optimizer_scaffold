# UI e renderização

## Stack sugerida
- Dash
- Dash AG Grid
- Dash Cytoscape
- Plotly

## Telas

### Tela 1 — Dados
- editar/importar/exportar tabelas
- catálogo de componentes
- regras de score
- requisitos de rota
- layout

### Tela 2 — Execução
- escolher cenário
- escolher famílias ativas
- parâmetros de geração
- iniciar geração/avaliação

### Tela 3 — Catálogo de soluções
- tabela com filtros
- colunas principais:
  - id
  - família
  - viável
  - custo
  - score
  - vazão de saída
  - fallback
  - robustez
  - limpeza

### Tela 4 — Comparação
- comparar 2 a 4 soluções
- radar/bar chart por métrica
- diferenças de BOM

### Tela 5 — Circuito
- renderização 2D com coordenadas
- destaque de bombas, medidores, válvulas e caminho por rota
- clique abre detalhes do componente

### Tela 6 — Escolha final
- solução selecionada
- justificativa por score
- export de BOM
- export de relatório
