# T04 — Geração de topologias

## Objetivo
Gerar topologias candidatas livres a partir de famílias e mutações.

## Entregáveis
- famílias base: star, bus, loop, hybrid
- mutações livres
- reparo/normalização
- persistência dos candidatos

## Critério de pronto
Catálogo inicial de candidatos gerado para a maquete.
