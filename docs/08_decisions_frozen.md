# Decisões congeladas

1. Superestrutura em camadas é obrigatória no MVP.
2. Operações são sequenciais.
3. Água (`W`) é fonte pura.
4. Saída (`S`) é sumidouro final.
5. Medição é direta em rotas de dosagem.
6. Cada rota usa exatamente uma bomba e um medidor/bypass.
7. Limpeza entre operações existe, mas entra como custo fixo no MVP.
8. Hidráulica é linearizada por perdas equivalentes em L/min.
9. Bitola é tratada por classe de sistema.
10. Adaptadores são materiais explícitos ou embutidos em opções.

Essas decisões não devem ser alteradas pelo agente de implementação sem aprovação humana.
