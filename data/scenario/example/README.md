# Cenário-exemplo

Este diretório contém um cenário mínimo para o scaffold.

## Arquivos
- `nodes.csv`
- `routes.csv`
- `components.csv`
- `source_branch_templates.csv`
- `destination_branch_templates.csv`
- `trunk_templates.csv`
- `settings.yaml`

## Observação
Os dados são ilustrativos e servem como base de desenvolvimento. O preprocessador e o solver
poderão evoluir a semântica exata dos templates conforme a implementação avançar.
